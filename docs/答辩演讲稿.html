答辩演讲稿 · 约 3 分钟

异步密语花园

Cipher Garden

第 1 分钟：我做了什么

「我的作品叫《异步密语花园》，是一个网页互动艺术作品——让朋友通过种花、解谜、扫码分享，在猜对之后才解锁隐藏密语的互动体验。」

「创作者在 5×5 花圃里选择 emoji、旋转角度和花色种花，可以藏一句文字，也可以拍一张照片当密语。生成链接或二维码发给朋友后，对方看不到答案，只能根据花的形态去猜。猜对有 Wordle 式反馈，一共 3 次机会；全部猜对后，花朵会绽放，密语才会出现。」

［现场演示 10 秒］
  • 快速种 2～3 格花 → 点「生成分享链接」→ 展示二维码
  • 或切换到解谜页，提交一次猜测，展示反馈颜色

第 2 分钟：为什么做 + 怎么做

「我想表达的是：在社交媒体里，表达往往太直接了；我想把『直接告诉』变成『猜对了才看见』——秘密被编码、传递、解锁的过程，本身就有仪式感和情绪张力。」

「用户通过种花、藏密语、扫码分享、解谜猜测互动，作品会生成一座独一无二的 parametric SVG 花圃，给出 Wordle 式逐格反馈，并在全部猜对后绽放揭晓秘密。」

「技术上我使用了 HTML、CSS、JavaScript，纯前端部署在 GitHub Pages；谜题数据用 lz-string 压缩嵌入 URL，配合 二维码方便微信扫码；照片密语用到 摄像头 API，大图则自动上传临时图床以保证链接可分享。」

第 3 分钟：最满意 + 未来计划

「我最满意的是『揭晓瞬间』——朋友猜了几次、终于全对的那一刻，花圃绽放、照片或文字缓缓出现，比直接发一张图或一句话更有记忆点。」

「如果继续迭代，我会做一个公开谜题墙让大家互相挑战，加入更多音视频密语形式，也会优化手机端的操作体验。」

「谢谢，欢迎体验。」
